# AI-Powered-Patient-Assistant
The AI-Powered Patient Assistant is an Agentic AI healthcare platform designed to improve patient engagement and healthcare accessibility through intelligent automation. The system leverages multiple specialized AI agents working collaboratively to assist patients 
# 🏥 AI-Powered Patient Assistant

Agentic AI Healthcare Platform built using:

- CrewAI
- LangChain
- GPT-4
- FAISS
- Streamlit
- Healthcare APIs

## Features

✅ Symptom Assessment

✅ Appointment Scheduling

✅ Medical FAQ Assistant

✅ Insurance Guidance

✅ RAG Knowledge Retrieval
